# Internship
## ICT 인턴십 관리자페이지 제작


### 소요기간 : 2 month
### - 사용 스택 : Javascript, Java, mariadb
### - 구현 내용 : 사이드바, 차트구현
### - 미구현 내용 : 로그인, 회원가입, 비밀번호 찾기, Anaylitcs Api 연동
### - 착안점 : 비즈니스 로직 향상을 위한 개발자 관리자 페이지 작업, 사용자 편의 UI제공
### - 아쉬운 점 : 미구현 페이지 구현 및 하드코딩
