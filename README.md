# Internship
## ICT 인턴십 회사 랜딩페이지 제작  
### -소요기간 : 2 month 
### -사용 스택 : Javascript, Node.js, TailWind, mongoDB
### -구현 내용 : 로그인, 회원가입, 회사 솔루션 홍보 컨텐츠 작성 및 기입 
### -미구현 내용 : 카카오소셜로그인 (리아디렉트 문제), 구매페이지 기능 (구독 기능) 
### -착안점 : 솔루션 장점 및 홍보 전략 구상 및 사용자 편의 고려한 사용자 친화적 홈페이지 작성 
### -아쉬운 점 : 컨설팅페이지 디자인 수정 필요, 미구현 내용 업데이트 필요 
### -배운 점 : 랜딩페이지에서 가장 중요한 점 ( 사용자가 처음 들어왔을 때 액션을 취하는 CTA)고려 및, 상세 디테일 부분 ( 어떠한 점을 사용자가 불편해할 수 도 있는 지 ) , 보여져야 할 컨텐츠 구상 및 배치방법 이해
